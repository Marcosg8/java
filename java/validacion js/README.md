Proyecto básico: Formulario de Evaluación de Sostenibilidad Personal

Instrucciones:
1. Coloca estos archivos en:
   c:\xampp\htdocs\marcos 2 año desarrollo no tocar pls\Marcos\java\validacion js

2. Si quieres ver la parte PHP, usa XAMPP y abre:
   http://localhost/marcos 2 año desarrollo no tocar pls/Marcos/java/validacion js/index.html
   (Ajusta la URL según tu configuración de htdocs.)

3. Archivo principales:
   - index.html : formulario y estructura semántica.
   - styles.css : estilos básicos y responsive.
   - validacion.js : validación cliente y cálculo simple.
   - procesar.php : validación servidor y resultado.
   - README.md : esta nota.

Notas:
- Esta es una versión básica para propósitos educativos. Para producción añade protección CSRF, validaciones más fuertes y almacenamiento seguro.
